# Instagram Downloader

> One page. Prose with bullets.

## What this is

인스타그램 포스트 URL을 입력하면 해당 게시물의 이미지와 동영상을 모두 다운로드하는 도구. 모바일 친화 web UI와 CLI 두 가지 인터페이스 제공.

## The problem

Personal need. 공개 인스타 포스트의 미디어를 손쉽게 받고 싶음. 모바일에서도 쓸 수 있어야 하고, 다른 자동화 에이전트가 호출할 수 있는 CLI도 필요.

## MVP — what v1 must do

- **공개 포스트 URL → 미디어 다운로드**: 단일 이미지, 캐러셀(다중), 비디오 모두 지원
- **모바일 친화 Web UI**: GitHub Pages에 정적 배포. URL 붙여넣기 → 미디어 미리보기/다운로드 버튼
- **CLI**: `instagram-dl <url>` 형태. 에이전트가 활용할 수 있도록 `--json` 출력 옵션과 명확한 exit code 제공

## Tech

- **Language**: TypeScript — CLI와 Web UI가 스크래핑 로직을 공유하기 위해 단일 언어
- **Runtime**: Node.js 20+ (CLI), 브라우저 (Web)
- **Web build**: Vite (정적 SPA)
- **Data**: 없음. 다운로드는 사용자 디바이스로 직접
- **Deploy**: Web → GitHub Pages (GitHub Actions). CLI → npm 또는 `npx` 직접 실행

## Out of scope

Explicitly NOT in v1:
- **인증/쿠키 기반 스크래핑** — 비공개 계정, 스토리, 릴스 중 로그인 필요한 것 제외
- **자체 백엔드 서버** — GH Pages 정적 호스팅 제약 안에서 해결. CORS 프록시 의존
- **배치/대량 다운로드, 큐, 히스토리, 즐겨찾기**
- **재인코딩, 워터마크 제거 등 미디어 후처리**

## Done means

공개 인스타 포스트 URL을 모바일 브라우저(GH Pages 배포본)에 붙여넣으면 미디어가 표시되고 저장 가능. 같은 URL을 CLI에 넘기면 미디어 파일이 디스크에 떨어지고 `--json`으로 메타데이터를 받을 수 있음.
